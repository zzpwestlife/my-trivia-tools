# My Trivia Tools

This repository contains a collection of useful little tools and utilities.

## Tools

*   **[Web Content Extractor](web-content-extractor/)**: A tool to extract web content (text, images) and convert it to Markdown.

## Usage

Navigate to the directory of the tool you want to use and follow the instructions in its `README.md`.

```bash
cd web-content-extractor
make install
make run
```
